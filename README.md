# Vicus
R package for graph embedding by Vicus method

## Installation

======
~~~~
git clone https://github.com/rikenbit/Vicus/
R CMD INSTALL Vicus
~~~~
or type the code below in the R console window
~~~~
library(devtools)
devtools::install_github("rikenbit/Vicus")
~~~~
